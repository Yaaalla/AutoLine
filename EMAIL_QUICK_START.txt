   └─ رسمية وحترافية

💡 محتوى رسالة العميل:

📋 البيانات المرسلة:

1. معلومات العميل:

   ✓ رقم الهاتف

2. تفاصيل السيارة:
   ✓ النوع (ماركة/موديل)
   ✓ ناقل الحركة
   ✓ عدد المقاعد
   ✓ نوع الوقود

3. فترة الحجز:
   ✓ تاريخ الاستلام
   ✓ تاريخ العودة
   ✓ عدد الأيام

4. التسعير:
   ✓ السعر اليومي
   ✓ عدد الأيام
   ✓ المبلغ الإجمالي

5. التواصل:
   ✓ رقم WhatsApp
   ✓ البريد الإلكتروني
