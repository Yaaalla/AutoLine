# 🚀 تعليمات سريعة - البدء الفوري

## ✨ ما تم إضافته؟

عندما يملأ العميل نموذج الحجز:
1. ✅ يتم حفظ البيانات
2. ✅ يتم إنشاء رسالة WhatsApp تلقائية
3. ✅ يظهر زر أخضر "متابعة الحجز عبر WhatsApp"
4. ✅ عند الضغط يفتح WhatsApp برسالة جاهزة

---

## ⚡ البدء الفوري (3 خطوات فقط)

### خطوة 1: افتح ملف الإعدادات
```
config/whatsapp_config.php
```

### خطوة 2: غيّر الرقم
**من:**
```php
define('WHATSAPP_NUMBER', '201001234567');
```

**إلى رقمك (مثال):**
```php
define('WHATSAPP_NUMBER', '201095555555');
```

**صيغة صحيحة:**
- 🇪🇬 مصر: `201` + الرقم = `201095555555`
- 🇸🇦 السعودية: `966` + الرقم = `966501234567`
- 🇦🇪 الإمارات: `971` + الرقم = `971501234567`

### خطوة 3: احفظ واختبر! ✅

---

## 🧪 الاختبار

1. اذهب: `car_details.php?id=1`
2. اضغط: "احجز هذه السيارة"
3. ملأ الفورم
4. اضغط: "تأكيد طلب الحجز"
5. اضغط: "متابعة الحجز عبر WhatsApp"
6. 🎉 يفتح WhatsApp!

---

## 💬 رسالة مثال

سيرسل العميل رسالة مثل هذه:

```
*طلب حجز سيارة جديد* 🚗

*بيانات العميل:*
👤 الاسم: محمد سالم
📧 البريد: customer@email.com
📱 الهاتف: +201001234567

*تفاصيل الحجز:*
🏎️ السيارة: BMW M5
📅 الاستلام: 15/03/2026
📅 العودة: 18/03/2026
⏱️ عدد الأيام: 3 يوم
💰 السعر اليومي: $150
💵 الإجمالي: $450
```

## ⚠️ مشاكل شائعة

| المشكلة | الحل |
|--------|------|
| لا يفتح WhatsApp | تأكد من تثبيت التطبيق |
| رسالة خاطئة | تحقق من الرقم بدون `+` أو مسافات |
| رقم غير صحيح | استخدم صيغة: كود الدولة + الرقم |
