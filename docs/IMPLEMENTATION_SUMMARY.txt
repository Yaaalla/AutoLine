  3. يظهر زر "متابعة الحجز عبر WhatsApp"
  4. عند الضغط يفتح WhatsApp برسالة جاهزة للإرسال

⚙️ خطوات الإعداد:

الخطوة 1️⃣: تعديل رقم WhatsApp
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
افتح: config/whatsapp_config.php

غيّر السطر:
   define('WHATSAPP_NUMBER', '201001234567');
   
إلى رقمك الفعلي بصيغة:
   الكود الدولي + الرقم (بدون + أو -)
   
أمثلة:
   🇪🇬 مصر:      201001234567
   🇸🇦 السعودية: 966501234567
   🇦🇪 الإمارات: 971501234567

الخطوة 2️⃣: اختبار الميزة
━━━━━━━━━━━━━━━━━━━━━━━━
1. اذهب إلى صفحة تفاصيل سيارة
2. اضغط "احجز هذه السيارة"
3. ملأ البيانات والتواريخ
4. اضغط "تأكيد طلب الحجز"
5. اضغط "متابعة الحجز عبر WhatsApp"
6. تحقق من WhatsApp
