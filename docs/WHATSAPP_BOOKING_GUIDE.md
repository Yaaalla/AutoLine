# دليل إعداد ميزة الحجز عبر WhatsApp

## الوصف
تم إضافة ميزة جديدة تسمح للعملاء بإرسال بيانات الحجز مباشرة إلى رقم WhatsApp الخاص بك بعد ملء نموذج الحجز.

## خطوات الإعداد

### 1. تعديل رقم WhatsApp
افتح ملف `booking_flow.php` وابحث عن السطر التالي:

```php
$whatsapp_number = "201001234567"; // مثال: رقم مصري
```

**استبدل الرقم بـ رقمك الفعلي:**
- صيغة الرقم: `<country_code><number>` (بدون علامات أو مسافات)
- مثال مصري: `201001234567` (20 = كود مصر، 1001234567 = الرقم)
- مثال سعودي: `966501234567` (966 = كود السعودية)

### 2. كيفية الاستخدام
1. العميل يملأ نموذج الحجز (الاسم، البريد، الهاتف، التواريخ)
2. عند الضغط على "تأكيد طلب الحجز"
3. يتم حفظ البيانات في قاعدة البيانات
4. يظهر للعميل رسالة نجاح مع زر "متابعة الحجز عبر WhatsApp"
5. عند الضغط على الزر، يتم فتح WhatsApp مع رسالة تحتوي على:
   - بيانات العميل (الاسم، البريد، الهاتف)
   - تفاصيل الحجز (السيارة، التواريخ، عدد الأيام)
   - السعر الإجمالي

## محتوى الرسالة
الرسالة تحتوي على:
```
*طلب حجز سيارة جديد* 🚗

*بيانات العميل:*
👤 الاسم: [اسم العميل]
📧 البريد: [بريد العميل]
📱 الهاتف: [رقم العميل]

*تفاصيل الحجز:*
🏎️ السيارة: [ماركة وموديل السيارة]
📅 الاستلام: [تاريخ الاستلام]
📅 العودة: [تاريخ العودة]
⏱️ عدد الأيام: [عدد الأيام]
💰 السعر اليومي: [السعر اليومي]
💵 الإجمالي: [السعر الإجمالي]
```

## الملفات المعدلة
- `booking_flow.php` - تم إضافة منطق إنشاء رابط WhatsApp

## ملاحظات هامة
1. **يجب تفعيل WhatsApp Business** على الرقم (اختياري) للحصول على ميزات أفضل
2. **يجب أن يكون لديك تطبيق WhatsApp مثبتاً** على الجهاز عند الضغط على الرابط
3. **الرسالة تفتح في محادثة جديدة** إذا لم تكن قد تحدثت مع الرقم من قبل
4. **يمكن حفظ الرقم مسبقاً** في جهات الاتصال لتسهيل التواصل

## اختبار الميزة
1. اذهب إلى صفحة تفاصيل أي سيارة
2. اضغط على "احجز هذه السيارة"
3. ملأ نموذج الحجز بالبيانات
4. اضغط على "تأكيد طلب الحجز"
5. اضغط على "متابعة الحجز عبر WhatsApp"
6. سيفتح WhatsApp بالرسالة الجاهزة للإرسال

## تخصيصات مستقبلية يمكن إضافتها
- إضافة رقم WhatsApp مختلف لكل فئة سيارات
- إرسال الرسالة تلقائياً (باستخدام Twilio API)
- حفظ سجل المحادثات
- إضافة أزرار سريعة للأسئلة الشائعة
