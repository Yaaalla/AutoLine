📊 **1. لوحة التحكم الرئيسية (Dashboard)**

ماذا تجد هنا؟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 6 بطاقات إحصائيات رئيسية:
   1️⃣ إجمالي الحجوزات - عدد كل الحجوزات
   2️⃣ قيد الانتظار - حجوزات لم تُؤكد بعد (اضغط لتصفيتها)
   3️⃣ مؤكدة - حجوزات تم تأكيدها (اضغط لتصفيتها)
   4️⃣ مكتملة - حجوزات انتهت (اضغط لتصفيتها)
   5️⃣ إجمالي الإيرادات - المبلغ الكلي
   6️⃣ متوسط الحجز - معدل قيمة الحجز

✅ رسم بياني دائري:
   - يعرض توزيع الحجوزات حسب الحالة
   - ألوان مختلفة لكل حالة
   - يتحدث تلقائياً عند إضافة حجوزات جديدة

✅ أفضل 5 سيارات:
   - السيارات الأكثر حجزاً
   - الإيرادات من كل سيارة
   - عدد الحجوزات لكل سيارة

✅ آخر 5 حجوزات:
   - أحدث حجوزات تم إضافتها
   - بيانات العميل والسيارة
   - الحالة والمبلغ

📋 **2. إدارة الحجوزات (Bookings)**

ماذا تجد هنا؟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ تصفية متقدمة:
   📌 اختر الحالة: كل الحالات، قيد الانتظار، مؤكد، مكتمل، ملغي
   📌 اختر التاريخ: اختياري - لتصفية بتاريخ معين
   📌 اضغط "تطبيق" لتطبيق التصفية
   📌 "إعادة ضبط" لحذف التصفية

✅ جدول الحجوزات:
   - المعرف والتاريخ
   - بيانات العميل (الاسم، البريد، الهاتف)
   - صورة ومعلومات السيارة
   - المبلغ الإجمالي
   - الحالة بألوان مختلفة
   - أزرار التحكم

✅ أزرار الإجراءات:
   🔵 عرض التفاصيل (عين) - اذهب للتفاصيل الكاملة
   ✅ تأكيد (علامة صح) - إذا كانت قيد الانتظار
   ✔️ اكتمال (علامة صح مزدوجة) - إذا كانت مؤكدة
   ❌ إلغاء (إكس) - لإلغاء الحجز

✅ رسائل التنبيه:
   🟢 أخضر: عملية نجحت
   🔴 أحمر: حدث خطأ

📄 **3. تفاصيل الحجز (Booking Details)**

ماذا تجد هنا؟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ حالة الحجز:
   - رقم الحجز والحالة
   - تاريخ إنشاء الحجز

✅ بيانات العميل:
   - الاسم الكامل
   - البريد الإلكتروني
   - رقم الهاتف

✅ بيانات السيارة:
   - الصورة
   - الماركة والموديل
   - نوع ناقل الحركة
   - عدد المقاعد
   - نوع الوقود
   - السعر اليومي

✅ فترة الحجز:
   - تاريخ الاستلام
   - تاريخ العودة
   - عدد الأيام

✅ التسعير:
   - السعر اليومي
   - عدد الأيام
   - الإجمالي النهائي

✅ الأزرار:
   ⬅️ العودة - للرجوع لقائمة الحجوزات
   🖨️ طباعة - لطباعة أو حفظ كـ PDF

🎨 **فهم الألوان والرموز**

الحالات:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🟡 أصفر = قيد الانتظار (Pending) - انتظر تأكيد العميل
🔵 أزرق = مؤكد (Confirmed) - تم تأكيد الحجز
🟢 أخضر = مكتمل (Completed) - الحجز انتهى
🔴 أحمر = ملغي (Cancelled) - تم إلغاء الحجز

الأيقونات الرئيسية:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📊 Dashboard = لوحة التحكم (الإحصائيات)
🚗 السيارات = إدارة الأسطول
📅 الحجوزات = إدارة الحجوزات
👥 المديرين = إدارة حسابات المديرين
🔙 عودة = الرجوع للصفحة السابقة
🖨️ طباعة = طباعة أو حفظ المستند

═══════════════════════════════════════════════════════════

⚡ **نصائح سريعة**

1. التحديث التلقائي:
   ✔️ البيانات تتحدث تلقائياً - لا تحتاج لتحديث يدوي

2. البحث السريع:
   ✔️ استخدم التصفية لإيجاد حجوزات معينة بسرعة

3. الطباعة:
   ✔️ يمكنك حفظ التفاصيل كـ PDF أثناء الطباعة

4. الجوال:
   ✔️ اضغط على الزر ☰ الموجود أعلى اليسار لفتح/إغلاق القائمة

5. الاتصال السريع:
   ✔️ انقر على أي رقم هاتف لاتصال مباشر (على الجوال)
   ✔️ انقر على البريد الإلكتروني لفتح تطبيق البريد

🔐 **معلومات الأمان**

✅ كل البيانات محمية ضد الهجمات:
   - استخدام Prepared Statements
   - فحص صلاحيات المسئول
   - تشفير البيانات الحساسة

✅ لا تشارك كلمة السر مع أحد:
   - كل مسئول له حساب خاص به
   - سجل الخروج دائماً عند الانتهاء

❓ **أسئلة شائعة**

س: كيف أضيف حجز جديد؟
ج: اضغط على زر "حجز جديد" في صفحة الحجوزات

س: كيف أغير حالة الحجز؟
ج: من جدول الحجوزات، اختر الإجراء المناسب (تأكيد، اكتمال، إلغاء)

س: كيف أطبع الحجز؟
ج: اذهب لتفاصيل الحجز واضغط "طباعة"

س: كيف أحذف حجز؟
ج: اختر الحجز وغير حالته لـ "ملغي" (لا يتم الحذف النهائي)

س: البيانات لا تتحدث؟
ج: أعد تحميل الصفحة بـ F5 أو Ctrl+Shift+R
