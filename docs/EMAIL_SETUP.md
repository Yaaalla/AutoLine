# 📧 خاصية إرسال البريد الإلكتروني - AutoLine

## 📋 الوصف

تم إضافة خاصية إرسال البريد الإلكتروني التلقائي عند استقبال طلب حجز جديد:

1. **بريد للعميل** - يحتوي على تفاصيل الحجز الكاملة
2. **بريد للمسؤول** - إشعار بالطلب الجديد

---

## ⚙️ الإعداد

### 1. تعديل البريد الإلكتروني الخاص بك

افتح الملف: `config/email_config.php`

غيّر البيانات التالية:

```php
// البريد الإلكتروني المستقبل (المسؤول)
define('ADMIN_EMAIL', 'your-email@example.com');

// البريد الإلكتروني للموقع (من المرسل)
define('FROM_EMAIL', 'booking@autolux.com');

// رقم الهاتف
define('CONTACT_PHONE', '+20 12 23789179');
```

---

## 📧 محتوى الرسائل

### الرسالة المرسلة للعميل:

```
التحية: مرحباً [اسم العميل]
المحتوى:
- بيانات العميل (الاسم، البريل، الهاتف)
- تفاصيل السيارة (النوع، ناقل الحركة، المقاعد)
- فترة الحجز (من/إلى، عدد الأيام)
- التسعير (السعر اليومي والإجمالي)
- معلومات التواصل
```

### الرسالة المرسلة للمسؤول:

```
الموضوع: طلب حجز جديد - [اسم السيارة]
المحتوى:
- بيانات العميل
- السيارة المطلوبة
- فترة الحجز
- المبلغ الإجمالي
- تاريخ ووقت الطلب
```

## 🔧 كيفية الاستخدام

العملية تحدث **تلقائياً** عند الضغط على "تأكيد طلب الحجز":

1. ✅ يتم حفظ البيانات في قاعدة البيانات
2. ✅ يتم إرسال بريد تأكيد للعميل
3. ✅ يتم إرسال إشعار للمسؤول
4. ✅ يفتح WhatsApp للمتابعة

## ⚠️ ملاحظات مهمة

1. **تفعيل البريد:**
   - تأكد من أن خادمك يدعم دالة `mail()`
   - في بعض الخواروم قد تحتاج لتفعيل SMTP

2. **البريد قد يصل للـ Spam:**
   - في بعض الأحيان قد تصل الرسائل إلى مجلد Spam
   - أضفنا headers صحيحة لتقليل هذه المشكلة

3. **اختبر الإرسال:**
   - قم بحجز تجريبي للتأكد من وصول البريد
